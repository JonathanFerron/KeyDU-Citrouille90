# PWM pins information for LED dimming

| ----- | ----- | -------- | -------- | -------- | -------- | -------- | -------- |
| 0x0   | PORTA | PA0 (30) | PA1 (31) | PA2 (32) | PA3 (1)  | PA4 (2)  | PA5 (3)  |
| 0x2   | PORTC | -        | -        | -        | PC3 (9)  | -        | -        |
| 0x3   | PORTD | PD0 (10) | PD1 (11) | PD2 (12) | PD3 (13) | PD4 (14) | PD5 (15) |
| 0x5   | PORTF | PF0 (20) | PF1 (21) | PF2 (22) | PF3 (23) | PF4 (24) | PF5 (25) |
